# spring_http

### HTTP method 별 Spring boot 기본 사용 format 정리

- GET
- POST
- PUT
- DELETE

### 다양한 Request 처리 방법 

- Request Parameter
  - Map
  - Dto
- Request Body
  - Json
- Path Variable

### 다양한 Response 처리 방법

- Response body
  - Text
  - Json
- HTML

### ETC
- Request, Reponse 변수 처리 시 형식 맞추는 법(Camel, Snake..)
- Object Mapper 변환 시 필요한 것들 (기본 생성자, get method)
